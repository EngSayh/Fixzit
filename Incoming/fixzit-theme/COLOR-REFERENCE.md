```css

  --color-primary: #25935F;
  --color-primary-hover: #188352;
  --color-primary-active: #166A45;
  --color-primary-light: #E8F7EE;
  
  /* Secondary */
  --color-secondary: #80519F;

  --color-bg: #F9FAFB;
  --color-bg-card: #FFFFFF;
  --color-text: #0D121C;
  --color-text-muted: #6C737F;

  --color-success: #17B26A;
```

## ‚ú® Quick Start

1. Copy `tailwind.config.ts` to your project root
2. Copy `globals.css` to your `app/` or `styles/` folder
3. Copy `theme.constants.ts` to your `lib/` folder
4. Copy `fonts.config.ts` to your `lib/` folder
5. Import fonts in your `layout.tsx`
6. Start building! üöÄ
